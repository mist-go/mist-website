**Mist** is a systems programming language that compiles directly into Rust.

It was designed to make high-performance development feel more direct and less mentally taxing, without compromising on the safety, performance, or ecosystem richness that Rust provides.

Mist is not a replacement for Rust. Instead, it serves as a more ergonomic surface layer for writing Rust-compatible systems code. It allows you to focus on the "what" while the compiler handles the "how" of idiomatic Rust generation.

---

## Philosophy: Systems Programming, Refined

Mist is built on the belief that systems programming should feel **calm, predictable, and easy to reason about**.

When working close to the metal, developers must balance performance, correctness, and structure. As projects scale, the "expression overhead"—the mental energy required to satisfy complex syntax—often creates friction. Mist reduces this friction by:

* **Focusing on Flow:** Code should feel direct to write and require minimal "re-contextualization" when revisited months later.


* **Reducing Mental Friction:** By simplifying the surface syntax, your attention stays on solving the problem, not fighting the mechanics of expression.


* **Predictable Abstractions:** Every feature in Mist is designed to be transparent, ensuring you always understand the underlying system behavior.



---

## What Mist Is

Mist is a compiled-to-Rust language specifically engineered for developers who need Rust's power but want a leaner authoring experience.

### Key Characteristics:

* **Native Ecosystem Integration:** Mist integrates directly with Cargo and existing Rust crates.


* **Idiomatic Output:** It compiles into clean, readable Rust code rather than opaque machine logic.


* **Zero-Cost Performance:** There is no runtime, no garbage collector, and no performance penalty.


* **Grounded Safety:** Mist remains fully powered by Rust’s type system and execution model.



Mist is especially valuable for large-scale codebases where readability is paramount, but the strict guarantees of a systems language are non-negotiable.
